### <a name="dateSQL"></a>bbn.fn.dateSQL(v, dayOnly)

  __Returns a date with SQL format.__

  * __v__ _Date|String_ 
  * __dayOnly__ _Boolean_ Whether or not include the time in the date

  __Returns__ _String_ 

### Examples



``` javascript
//"2020-04-16 16:15:23"
let date = new Date();
bbn.fn.dateSQL(date,false);
```
