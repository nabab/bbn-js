### <a name="br2nl"></a>bbn.fn.br2nl(st)

  __Replaces the html <br> tag with new line characters '\ n' if present in the string.__

  * __st__ _trin_ 

  __Returns__ _String_ 

### Examples



```javascript
//"hello
//world!"
bbn.fn.br2nl('hello <br> world!')
```
