### <a name="date"></a>bbn.fn.date(v)

  __Returns a date object from the given argument.__

  * __v__ _String|Number_ 

  __Returns__ _date_ 

### Examples



``` javascript
//Mon Feb 11 2019 12:00:00 GMT+0100 (Central European Standard Time)
bbn.fn.date('2019/02/11')
```
