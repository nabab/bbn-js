### <a name="timestamp"></a>bbn.fn.timestamp(seconds)

  __Returns the timestamp of the given seconds if an argument is given, else returns the timestamp of new Date().__

  * __seconds__ _Boolean_ 

  __Returns__ _Number_ 

### Examples



```javascript
//1587031047918
bbn.fn.timestamp();
```
