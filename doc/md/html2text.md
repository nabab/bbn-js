### <a name="html2text"></a>bbn.fn.html2text(st)

  __Convert text in html format to plain text.__

  * __st__ _String_ 

  __Returns__ _String_ 

### Examples



```javascript
//"Hello world!"
bbn.fn.html2text("<div><p>Hello <b>world!</b></p></div>");
```
