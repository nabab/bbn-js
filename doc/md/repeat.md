### <a name="repeat"></a>bbn.fn.repeat()

  __Returns a string which is the repetition of the first argument for the number passed in the second argument.__


  __Returns__ _String_ 

### Examples



```javascript
//"HelloHelloHello"
bbn.fn.repeat('Hello', 3);
```
