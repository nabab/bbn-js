### <a name="isEmpty"></a>bbn.fn.isEmpty(obj)

  __Checks if the argument is empty or not.__

  * __obj__ _Mixed_ 

  __Returns__ _Boolean_ 

### Examples



```javascript
bbn.fn.isEmpty({});
//true
```


```javascript
bbn.fn.isEmpty({test : 1});
//false
```


```javascript
bbn.fn.isEmpty([]);
//true
```


```javascript
bbn.fn.isEmpty(['test']);
//false
```


```javascript
bbn.fn.isEmpty('');
//true
```


```javascript
bbn.fn.isEmpty('test');
//false
```
