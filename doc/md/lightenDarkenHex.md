### <a name="lightenDarkenHex"></a>bbn.fn.lightenDarkenHex()

  __Takes color in hex format and lightens or darkens it with the given value.__


  __Returns__ _String_ 

### Examples



```javascript
//"#eccb28"
bbn.fn.lightenDarkenHex('#c4a300', 40);
```



```javascript
//"#9c7b00"
bbn.fn.lightenDarkenHex(#c4a300', -40);
```
