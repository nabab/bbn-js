### <a name="isObject"></a>bbn.fn.isObject()

  __Returns true if the given argument is an object.__


  __Returns__ _Boolean_ 

### Examples



```javascript
bbn.fn.isObject({name: 'cami', age: 7});
//true
```


```javascript
bbn.fn.isObject([{name: 'cami', age: 7}]);
//false
```
