### <a name="isPrimitive"></a>bbn.fn.isPrimitive()

  __Returns true if the given arguments are primitive;.__


  __Returns__ _Boolean_ 

### Examples



```javascript
bbn.fn.isPrimitive('myString', 6, true);
//true
bbn.fn.isPrimitive([80,10,22]);
//false
bbn.fn.isPrimitive({});
//false
```
