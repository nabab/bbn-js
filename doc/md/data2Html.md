### <a name="nl2br"></a>bbn.fn.nl2br(st, clsName)

  __Replaces all new line characters '\ n' with html tag '<br>'.__

  * __st__ _Array!Object_ 
  * __[clsName]__ _String_ 

  __Returns__ _String_ 

### Examples



```javascript
bbn.fn.nl2br('hello \n world!');
//"hello <br> world!"
```
