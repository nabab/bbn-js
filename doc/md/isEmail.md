### <a name="isEmail"></a>bbn.fn.isEmail(st)

  __Intended to check if the argument provided is an e-mail address written correctly.__

  * __st__ _String_ 

  __Returns__ _Boolean_ 

### Examples



```javascript
bbn.fn.isEmail('test@testorg');
//false
```



```javascript
bbn.fn.isEmail('test@test.org');
//true
```
