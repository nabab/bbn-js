### <a name="isDate"></a>bbn.fn.isDate()

  __Returns true if the given argument is a date object.__


  __Returns__ _Boolean_ 

### Examples



```javascript
let date = new Date();
bbn.fn.isDate(date);
//true
```


```javascript
bbn.fn.isDate('16/04/2020');
//false
```
