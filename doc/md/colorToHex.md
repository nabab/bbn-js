### <a name="colorToHex"></a>bbn.fn.colorToHex()

  __Returns the hex color of the given rgb or color name.__


  __Returns__ _String_ 

### Examples



```javascript
//"#ff0000"
bbn.fn.colorToHex('red');
```



```javascript
//"#ff0000"
bbn.fn.colorToHex('rgb(255,0,0)');
```
