### <a name="isNotObject"></a>bbn.fn.isNotObject()

  __Returns true if the given argument is not null or type object or array.__


  __Returns__ _Boolean_ 

### Examples



```javascript
bbn.fn.isNotObject('myString');
//true
```


```javascript
bbn.fn.isNotObject(6);
//true
```


```javascript
bbn.fn.isNotObject([80,10,22]);
//false
```
