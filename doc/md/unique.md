### <a name="unique"></a>bbn.fn.unique(arr)

  __Removes duplicate values from an array.__

  Takes an input array and returns a new array without duplicate values.

  * __arr__ _Array_ 

  __Returns__ _Array_ 

### Examples



```javascript
bbn.fn.unique(["a", "b", "a", "b", "a", "b", "c", "c", "d"]);
// ["a", "b", "c", "d"]
```
