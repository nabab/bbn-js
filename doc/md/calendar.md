### <a name="calendar"></a>bbn.fn.calendar(d, |)

  __Returns a date relative to the current day.__

  * __d__ _Date|String_ 
  * __|__ _Strin_ Boolean} wrong_result Whether or not include the time in the date

  __Returns__ _String_ 

### Examples



``` javascript
//"2020-04-16 16:15:23"
let date = new Date();
bbn.fn.dateSQL(date,false);
```
