### <a name="isBoolean"></a>bbn.fn.isBoolean()

  __Returns true if the given argument is a boolean.__


  __Returns__ _Boolean_ 

### Examples



```javascript
const sb = true;
bbn.fn.isBoolean(sb); // true
const sb = 1;
bbn.fn.isBoolean(sb); // false
```
